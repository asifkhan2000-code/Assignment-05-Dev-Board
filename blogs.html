Question-1: What are the different ways to select an element in the DOM?

In JavaScript, you can select DOM elements using methods like getElementById() for selecting by ID, getElementsByClassName() for class-based selection, and getElementsByTagName() for tag-based selection. More modern and flexible methods include querySelector(), which selects the first element matching a CSS selector, and querySelectorAll(), which selects all matching elements as a NodeList.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets or sets the HTML inside an element, including tags. innerText gets or sets only the visible text, respecting CSS styles. textContent gets or sets all the text inside an element, ignoring HTML and CSS visibility.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a parent element handles events for its child elements. Instead of adding event listeners to multiple child elements, you add a single listener to their common parent. The event propagates up the DOM (bubbling), and the parent captures it, improving performance and handling dynamically added elements efficiently.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is a phase where an event starts from the target element and propagates upward through its parent elements up to the root. For example, if you click a button inside a div, the click event first fires on the button, then on its parent div, and continues up the DOM tree. This behavior allows techniques like event delegation.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, you create elements with document.createElement(), add them using methods like appendChild(), append(), or insertBefore(), and remove them with removeChild() or remove().
